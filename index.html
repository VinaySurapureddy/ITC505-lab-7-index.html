Create a Story!

Addendum

This lab implements a Mad Lib generator using an HTML form and a Node.js/ExpressJS backend. The form submits via POST to the same URL and dynamically generates a custom story using string interpolation.

A standalone /do_a_random/ route was implemented to serve a random number when accessed via browser directly (e.g., http://localhost:8080/do_a_random/), fulfilling the routing requirement.

Styling was kept minimal for clarity. Page verified at http://localhost:8080/ITC505/lab-7/index.html with all required features functional, including the footer script.
